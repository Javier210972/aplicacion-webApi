## Como ejecutar el proyecto
### Backend (ASP.NET Core Web API)
```bash
cd backend/Arithmetic.Api
```
```bash
dotnet run --urls "http://localhost:5273"
```

### FrontEnd (Angular)
```bash
cd frontend/arithmetic-app
```
```bash
Solo la primera vez: npm install
```
